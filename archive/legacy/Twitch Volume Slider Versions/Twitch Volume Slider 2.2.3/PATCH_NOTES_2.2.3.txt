📋 Patch Notes - Version 2.2.3

### 🐛 BUG FIX

- **Instant Press Snap**: Pressing the volume bar in free-drag mode now snaps immediately to the nearest 5% mark instead of moving to the exact pointer position first.
- **Stable Options Text Size**: The custom settings menu now uses fixed typography metrics so button text no longer shifts size on the first open after page load.
- **Stable On-Video Positioning**: The on-video slider now falls back to a fixed low position when native controls measurements are unavailable, preventing upward drift while hovering.
- **Twitch Controls Visibility Detection**: Twitch on-video positioning now follows the outer `player-controls` visibility state, so hidden controls are not mistaken for a visible controls area.
- **Native-Like Menu Hold**: The Twitch controls and custom settings menu now stay open when the mouse leaves the controls area, while outside clicks close the menu and keep the controls visible briefly like Twitch's native settings menu.

### 👀 VISUAL CHANGE

- **Cleaner Button Text Alignment**: Menu buttons now have explicit heights, line heights, and centered flex layout for more consistent text positioning.

### 🔧 CODE OPTIMIZATION

- **Press-Aware Slider Input**: Added a small first-input handler that preserves free dragging while matching the older 1.5.1 press behavior.

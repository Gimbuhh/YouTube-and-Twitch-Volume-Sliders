# Volume Sliders 2.2

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### ✨ NEW FEATURES

- **Dedicated Options Button**: Added a Volume Slider button next to YouTube's native settings button for opening the slider settings without using YouTube's settings menu.
- **Complete Custom Settings Panel**: Added one panel for On, Off, Replace native, replacement placement, Snap to 5%, Always expanded, Show on video, and on-video opacity.
- **On-Video Slider Mode**: Added an option to move the slider from the controls bar onto the video with separate idle and active opacity controls.

### 👀 VISUAL CHANGES

- **Cleaner Settings UI**: Replaced the native-menu integration with a compact custom panel that matches YouTube's dark player styling.
- **Centered Checkbox Checks**: Refined checkbox styling so selected settings look centered and polished.
- **Settings Button Placement**: Positioned the Volume Slider settings button directly beside YouTube's native settings button.
- **Progress Bar Clearance**: Positioned the options panel above YouTube's progress bar so it no longer covers the scrubber.

### 🛠️ CONFIGURATION UPDATES

- **Saved Setting Support**: Preserved saved settings for slider mode, placement, snap behavior, expanded behavior, video placement, and opacity.
- **Code Overrides Preserved**: Kept `USER_SETTINGS` overrides for users who prefer script-level defaults instead of saved browser settings.
- **Legacy Storage Removed**: Removed old unused migration keys so 2.2 relies only on the current settings model.

### 🐛 BUG FIXES

- **No Native Menu Conflicts**: Removed the old dependency on YouTube's native settings menu, avoiding submenu sizing, clipping, and stale state issues.
- **Stable Replace Native Mode**: Improved native volume replacement so the custom slider can occupy the intended controls position reliably.
- **Popup Auto-Close Behavior**: The custom settings panel closes cleanly on outside click, Escape, YouTube settings open, player control hide, and navigation.
- **Native-Like Preview Suppression**: Progress-bar preview thumbnails are hidden while the custom options panel is open, matching YouTube's native settings menu behavior.

### 🔧 CODE OPTIMIZATIONS

- **Removed Submenu Code**: Deleted the old native-settings submenu implementation and its sizing/refit helpers.
- **Simpler Settings Surface**: Consolidated all slider preferences into one custom popup flow.
- **Release Naming Cleanup**: Removed the WIP label from the YouTube 2.2 userscript name.

## Twitch

### ✨ NEW FEATURES

- **Dedicated Options Button**: Added a Twitch Volume Slider settings button beside the native player settings button.
- **Complete Custom Settings Panel**: Added On, Off, Replace native, replacement placement, Snap to 5%, Always expanded, Show on video, and on-video opacity controls.
- **Replace Native Mode**: Added a mode that hides Twitch's native volume control and places the custom slider in its spot.
- **Show On Video Mode**: Added an option to move the slider onto the video with adjustable idle and active opacity.

### 👀 VISUAL CHANGES

- **YouTube 2.2 Slider Style**: Updated the Twitch slider pill, bar, icon, and expansion behavior to match the newer YouTube 2.2 implementation.
- **Twitch Accent Styling**: Kept Twitch-specific purple accents for selected buttons, checkboxes, and the volume slider.
- **Neutral Settings Panel**: Updated the Twitch settings panel to use the same neutral dark panel style as YouTube, with purple only for selected controls.
- **Correct Button Placement**: Positioned the Volume Slider settings button after the quality indicator and clip button, directly before Twitch's native settings cog.

### 🛠️ CONFIGURATION UPDATES

- **Saved Setting Support**: Added saved settings for slider mode, replacement placement, snap behavior, expanded behavior, video placement, and opacity.
- **Code Overrides Added**: Added `USER_SETTINGS` overrides so script-level defaults can still be used instead of saved browser settings.

### 🐛 BUG FIXES

- **Stable Dragging In Controls**: Fixed the slider refreshing while dragging in Twitch's controls area.
- **Stable Dragging On Video**: Fixed the same reattach behavior when the slider is shown on the video.
- **No Rapid Collapse Loop**: Fixed the pill rapidly collapsing and expanding while changing volume.
- **Button Slot Self-Correction**: Fixed the settings button staying too far left when Twitch added the quality indicator or clip button after injection.
- **Native Replacement Visibility**: Fixed native volume hiding and custom slider placement in Replace native mode.

### 🔧 CODE OPTIMIZATIONS

- **Idempotent Overlay Placement**: Made Twitch overlay placement reuse the existing element instead of constantly moving or rebuilding it.
- **Interaction-Aware Reattach Logic**: Prevented Twitch DOM updates from interrupting hover, focus, or drag interactions.
- **Scoped Settings Lookup**: Scoped the Twitch settings-button lookup to the right controls group for more reliable placement.

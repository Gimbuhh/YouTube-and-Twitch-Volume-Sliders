> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### 🎨 NEW FEATURES

- **Volume Persistence**: Added localStorage support to remember volume settings across page reloads and navigation.
- **Smart Positioning**: Overlay now dynamically positions itself just above the video scrubber bar instead of fixed percentage.
- **Modern Slider Design**: Redesigned slider with thinner track (8px), modern thumb design, and smooth gradient fill effect.
- **Enhanced Hover Effects**: Added scale transform and improved shadow effects on slider thumb hover and active states.

### ⚡ PERFORMANCE IMPROVEMENTS

- **Improved Visual Transitions**: Upgraded overlay transitions with blur backdrop filter and smooth opacity/transform effects.
- **Adaptive Positioning**: Overlay automatically adjusts position based on scrubber bar location with 500ms delayed recheck.

### 🛠️ CONFIGURATION UPDATES

- **Updated Color Scheme**: Refined gradient from harsh red to smoother red tones (#ff2222 to #ff0000 to #ff3333).
- **Enhanced Overlay Styling**: Darker background (0.35 opacity), glass-morphism effect with 12px blur, and subtle white border.
- **Resize Handler**: Added window resize listener to maintain correct positioning across viewport changes.

## Twitch

### 🎨 NEW FEATURES

- **Visual Tick Marks**: Added 5% interval tick marks that appear on hover for precise volume adjustment.
- **Snap-to-Grid**: Quick clicks (under 300ms) now snap volume to the nearest 5% increment for easier control.
- **Mute Button**: Added dedicated mute/unmute button with visual icons for quick audio toggling.

### ⚡ PERFORMANCE IMPROVEMENTS

- **Optimized Sync Logic**: Removed polling interval and switched to event-driven volume sync for better performance.
- **Smooth Initial Load**: Overlay now fades in from opacity 0 instead of jumping to 0.45 on first load.
- **Storage Read Optimization**: Slider initializes from localStorage on creation to prevent visual jump from 100% to actual volume.

### 🔧 CODE OPTIMIZATION

- **Enhanced Error Handling**: Wrapped all API interactions in try-catch blocks to prevent script crashes when React structure changes.
- **Removed Slider Disabled State**: Simplified mute behavior by removing disabled state when muted.
- **Improved Volume Sync**: Changed volumechange listener to only update UI without saving, preventing feedback loops.

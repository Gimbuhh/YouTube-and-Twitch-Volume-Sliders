# Volume Sliders 1.3.2

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### 🎨 NEW FEATURE

- **Overlay Toggle Button**: Added a new speaker-style toggle button near the Like controls so you can quickly enable or disable the volume overlay.

### ⚡ PERFORMANCE IMPROVEMENT

- **Overlay Teardown When Disabled**: Disabling the toggle now removes the overlay and its listeners instead of only hiding it, reducing unnecessary UI work.

### 🛠️ CONFIGURATION UPDATE

- **Toggle Visibility Setting**: Added `showToggleButton` in `USER_SETTINGS` so the toggle button itself can be enabled or disabled.
- **Persistent Overlay State**: The overlay enabled/disabled state now saves between page loads, so your preference is remembered automatically.

### 🐛 BUG FIXES

- **Visual State Feedback**: Improved active/inactive button visuals so it is clear when the overlay is enabled versus disabled.
- **State Sync Reliability**: Button icon/state updates now stay in sync with the real overlay state during toggles and navigation changes.

## Twitch

### 🎨 NEW FEATURE

- **Overlay Toggle Button**: Added a new speaker-style toggle button beside channel action controls so you can quickly enable or disable the volume overlay.

### ⚡ PERFORMANCE IMPROVEMENT

- **Overlay Teardown When Disabled**: Disabling the toggle now removes the overlay and cleanup handlers instead of only hiding it, improving efficiency.

### 🛠️ CONFIGURATION UPDATE

- **Toggle Visibility Setting**: Added `showToggleButton` in `USER_SETTINGS` so the toggle button can be shown or hidden.
- **Persistent Overlay State**: The overlay enabled/disabled preference is now saved and restored automatically.

### 🐛 BUG FIXES

- **Follow/Unfollow Placement**: Improved insertion behavior so the toggle is reliably injected next to channel controls.
- **State & Icon Sync**: Updated active/inactive visuals and icon switching so toggle feedback stays consistent.

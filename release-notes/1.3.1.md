## YouTube

### 🐛 BUG FIX

## Twitch

### 🐛 BUG FIXES

- **Volume Sync on Page Load**: Fixed custom volume not applying correctly on page refresh, preventing the brief flash of native Twitch volume before custom value loads.
- **Two-Way Volume Sync**: Fixed native Twitch volume slider and custom overlay slider not staying synchronized or persisting the same value across page reloads.

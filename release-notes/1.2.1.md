## YouTube

### 🎨 NEW FEATURES

- **Softer Slider Colors**: Reduced slider color intensity with more muted red tones for a less visually aggressive appearance, especially when overlay is not in focus.

### 🛠️ CONFIGURATION UPDATES

- **Color Palette Adjustment**: Updated gradient from bright reds (#ff2222, #ff0000, #ff3333) to subdued tones (#cc4444, #bb3333, #cc5555) for improved visual comfort.

## Twitch

### 🐛 BUG FIX

- **Magnetic Snapping**: Fixed volume slider snapping behavior where clicking at a value (e.g., 57%) would briefly show that value before jumping to the nearest 5% increment (55%), now snaps directly to the target value for a smoother, more intuitive experience.

# Volume Sliders 1.4.2

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## Twitch

### 🐛 BUG FIXES

- **Volume Restore on Channel Switch**: Fixed saved volume not applying correctly when switching channels without a full page refresh (e.g. clicking a different streamer from the sidebar).

- **Volume Restore on Fast Machines**: Fixed a race condition where on fast computers — where the stream loads near-instantly — Twitch's player would reset the volume before the script had a chance to protect it, overwriting the saved value in storage and causing 100% volume on every channel switch.

- **Twitch Player Override During Load**: Fixed an issue where Twitch's internal player initialization could silently override the restored volume mid-load; the script now actively re-applies the saved value if it detects Twitch has changed it during the startup window.

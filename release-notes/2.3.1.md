# Volume Sliders 2.3.1

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### ⚡ PERFORMANCE IMPROVEMENTS

- **No More Background Polling**: Removed repeating timers that checked volume hold and options-menu state several times per second; the slider now reacts on a schedule only when needed and when player controls change.
- **Smaller DOM Watch Area**: Reattach detection now watches the YouTube player area instead of the entire page, so unrelated site updates trigger less work.
- **Faster Startup Attach**: Initial attach retries stop as soon as the slider is ready instead of running on a fixed interval until the attempt limit.

### 🐛 BUG FIXES

- **Observer Rebind After Navigation**: Player control observers now reconnect when YouTube replaces the player during navigation, so options-menu hold behavior stays reliable after clicking between videos.
- **Layout Guard During Drag**: On-video repositioning no longer interrupts while you are dragging or hovering the volume slider.

### 🔧 CODE OPTIMIZATIONS

- **Theme Color Consistency**: The volume bar gradient now uses the shared accent constant instead of a one-off hardcoded red.
- **Shared Snap Helper**: Slider snap logic uses the same helper on press and release for easier maintenance.

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### 🎨 NEW FEATURES

- **Mute Button**: Added dedicated mute/unmute button with animated SVG icons for quick audio control.
- **Visual Tick Marks**: Added 5% interval tick marks that appear on hover for precise volume adjustment.
- **Snap-to-Grid**: Quick clicks (under 300ms) now snap volume to the nearest 5% increment for easier control.
- **Muted State Display**: Label now shows "Muted" text when audio is muted.

### ⚡ PERFORMANCE IMPROVEMENTS

- **YouTube Player Caching**: Added caching for YouTube player API to reduce repeated DOM queries.
- **Video Element Caching**: Implemented video element caching to minimize querySelector calls.
- **Debounced Storage Writes**: Volume saves now debounced to 150ms during slider drag, reducing localStorage write frequency.
- **Optimized Mutation Observer**: Observer filters for only relevant mutations (added nodes) and uses requestAnimationFrame throttling.

### 🐛 BUG FIXES

- **Event Listener Cleanup**: Added proper cleanup for volumechange event listener to prevent memory leaks.
- **Storage Timer Cleanup**: Timer properly cleared when overlay is removed to prevent orphaned timeouts.
- **Navigation Handler Guard**: Prevented duplicate event binding with global flag check.
- **Improved Mute Sync**: Mute button icon updates via display style toggle only, avoiding MutationObserver feedback loops.
- **Smart Volume Restore**: Only restores saved volume on first attach and when not muted to respect user mute state.

### 🛠️ CONFIGURATION UPDATES

- **Accessibility Labels**: Added aria-label attributes to mute button and volume slider for screen reader support.
- **SVG Icon System**: Replaced innerHTML-based icons with proper SVG DOM construction to prevent mutation conflicts.
- **Debounce Constant**: Introduced STORAGE_WRITE_DEBOUNCE_MS constant (150ms) for configurable write delay.

## Twitch

### ⚡ PERFORMANCE IMPROVEMENTS

# Volume Sliders 2.5

## Added

- Added an on-video size control for YouTube and Twitch so the floating volume slider can be enlarged from 100% to 200%.
- Keeps the size setting scoped to the Show on video placement, with the control hidden and the overlay returned to normal size when the slider is in the controls.

## Changed

- Scales the full on-video volume pill together, including the icon, value label, slider rail, track, thumb, background, and hit targets.

# Volume Sliders 1.5.1

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### 🛠️ CONFIGURATION UPDATE

- **Overlay Transparency Setting**: Added `overlayOpacityUnfocused` and `overlayOpacityFocused` to USER_SETTINGS as whole-number percentages (0–100), defaulting to 45 and 95. Change these two values to control how transparent the overlay appears at rest and when hovered.

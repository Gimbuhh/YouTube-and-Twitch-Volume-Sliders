> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### 🎨 NEW FEATURE

- **Adaptive Overlay Positioning**: The volume slider now sits at the bottom of the video when YouTube controls are hidden, and moves above the progress bar when controls reappear on hover. Toggle with the `adaptivePosition` setting (default: `true`).

### 🔧 CODE OPTIMIZATIONS

- **`getSavedVolume()` Helper**: Extracted the repeated localStorage read/parse/clamp logic into a single reusable function shared across the script.

- **Slider Initializes from Saved Volume**: The slider now reads from localStorage on creation instead of the live player, preventing a brief jump from 100% to the actual volume on load.

- **Fixed Redundant Style Injection**: `createStylesIfNeeded` and `ensureToggleButtonStyles` now early-return if the style element already exists, instead of rewriting it on every call.

- **Merged MutationObservers**: Combined the slider reattach observer and toggle button re-injection observer into a single observer to reduce DOM mutation overhead.

- **Navigation Debouncing**: Reattach on YouTube page navigation is now debounced by 180ms and stale player/video caches are cleared on navigation, preventing stacked reattach cycles.

- **Cleaned `syncMuteButton` Signature**: Removed an unused parameter that was passed to the function but never referenced.

- **Inlined `showOverlay` / `hideOverlay`**: Removed two redundant one-liner wrapper functions by inlining their calls directly into `setOverlayEnabled()`.

- **Reduced Startup Polling**: Startup polling cut from 40 attempts (~20s) to 20 attempts (~10s), since the MutationObserver handles ongoing reattachment.

## Twitch

### 🐛 BUG FIXES

- **User Intent Protection**: Manual volume or mute changes are now protected for 5 seconds — the late-recovery restore will not override a volume you just intentionally set.

- **Single Startup Correction**: The startup volume re-apply now fires only once per attach cycle, preventing volume flicker on fast machines.

- **Navigation Reattach Guard**: Reattach now only triggers when the URL path actually changes, preventing unnecessary reattach cycles from Twitch's internal navigation calls that don't change the page.

- **Timer Stacking Prevention**: Navigation timers are now cancelled before new ones are scheduled, preventing overlapping timer chains on rapid channel switching.

### 🔧 CODE OPTIMIZATIONS

- **`getSavedVolume()` Helper**: Extracted the repeated localStorage read/parse/clamp logic — which appeared in 4 separate places — into a single reusable function.

- **Simplified Late Recovery Restore**: The late-restore block now delegates to `restoreSavedVolume()` and `setSliderFromPlayer()` instead of reimplementing their logic inline.

- **Removed Dead Positioning Code**: Removed `positionOverlayAtBottom`, its resize listener, and its cleanup — the overlay already sets `bottom: 12px` at creation, making these no-ops.

- **Fixed Redundant Style Injection**: `createStylesIfNeeded` and `ensureToggleButtonStyles` now early-return if the style element already exists, instead of rewriting it on every call.

- **Merged MutationObservers**: Combined the slider reattach observer and toggle button re-injection observer into a single observer to reduce DOM mutation overhead.

- **Inlined `showOverlay` / `hideOverlay`**: Removed two redundant one-liner wrapper functions by inlining their calls directly into `setOverlayEnabled()`.

- **Reduced Startup Polling**: Startup polling cut from 40 attempts (~20s) to 20 attempts (~10s), since the MutationObserver handles ongoing reattachment.

# Volume Sliders 2.2.1

> Historical release. The exact original release date was not recorded; this version was archived by 2026-03-29.

## YouTube

### 👀 VISUAL CHANGE

- **Consistent Slider Bar Thickness**: The volume slider bar now always uses the thicker hover-state height, so it no longer changes size when hovered.

### 🔧 CODE OPTIMIZATION

- **Simpler Slider Hover Styling**: Removed the hover-only track height change and made the thicker bar the default.
